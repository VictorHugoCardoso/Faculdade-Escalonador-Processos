Algoritmo 1: FCFS (Fisrt come, First served)
Algoritmo 2: Escalonamento por prioridade

Plataforma:
Sistema Operacional
- Windows 7,8,10 - Linux Ubuntu

Compilador
- gcc Debian 7.3.0


Informações para compilação e execução do programa:
Compilar
gcc main.c -o {output_name}
Ex: gcc main.c -o main 

Executar
./{output_name} {input_file_name} {output_file_name}
Ex: ./main entrada.txt

Informações arquivo de entrada:
Ex:
[1,2,3]
[2,3,2] <---- informações de casa processo obs: [id, surto, prioridade]
